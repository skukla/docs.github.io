## Create A Skeleton

<Steps>

1. Create a new folder on your desktop and give it a name. Don't use captial letters and make sure to use dashes in place of spaces. For example: `my-data-pack`

2. Add a `data/` folder and a `media/` folder to `my-data-pack`.

</Steps>

<FileTree>

    - my-data-pack/
      - **data/**
      - **media/**

</FileTree>
